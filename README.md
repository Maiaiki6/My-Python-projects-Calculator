# My-Python-projects-Calculator

A simple four-operation calculator

Simple Calculator

This is a small Python calculator that supports addition, subtraction, multiplication, and division.

Usage

Run the script and follow prompts:

```
python calculator.py
```

- Enter `1`-`4` to choose an operation.
- Enter numbers when prompted.
- Type `q` or `quit` at any prompt to exit.

Notes

- Division by zero is handled and prints an error message instead of crashing.
- The arithmetic functions are available for import: `add`, `subtract`, `multiply`, `divide`.
